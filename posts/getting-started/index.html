Getting Started with Typstify

January 20, 2024
tutorial guide

This guide will walk you through setting up your first Typstify site.

Installation

First, install Typstify using Cargo:

cargo install typstify

Creating a New Site

Create a new directory for your site:

mkdir my-site
cd my-site

Initialize the configuration:

typstify new config.toml

Writing Content

Markdown Posts

Create your first post:

typstify new posts/my-first-post

This creates a new Markdown file with frontmatter:

---
title: "My First Post"
date: 2024-01-20
draft: true
tags: []
---

Write your content here.

Typst Documents

For more complex documents, use Typst:

typstify new docs/technical-doc --template typst

Building Your Site

Build for production:

typstify build

Or start the development server:

typstify watch --open

Configuration

Edit config.toml to customize your site. See the configuration reference for all options.
