This is a sample blog demonstrating the capabilities of [Typstify](https://github.com/longcipher/typstify), a high-performance static site generator.

## About Typstify

Typstify is built with:

- **Rust** for blazing fast performance
- **Leptos** for reactive UI components
- **Tantivy** for full-text search
- **WASM** for client-side search

## Features Demonstrated

This example site showcases:

- Markdown content
- Typst documents
- Multi-language support (English & Chinese)
- Tags and taxonomies
- Full-text search
- RSS feed
- Sitemap generation
