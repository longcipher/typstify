## 为什么选择 Typstify？

Typstify 具有以下优势：

1. **极速构建** - 使用 Rust 构建，性能卓越
2. **双格式支持** - 支持 Markdown 或 Typst 编写
3. **内置搜索** - 基于 WASM 的全文搜索
4. **实时预览** - 开发时即时反馈
